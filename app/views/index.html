February 2025

Version 1

  • Record a grade, Start page

    • Changelog

    This version includes:

    • Creation of the initial prototype.
    • The framework to run both the GOV.UK Frontend and DfE Frontend in one prototype
    • The start page
    • The dashboard
    • All the questions pages
    • The 'Check your answers' page
      • This includes the 'shortcutting' functionality to skip questions if they've already been answered once
    • The success page

    What isn't included (yet):

    • Anything that would be outside of the 'Record a grade' journey
    • That includes the 'Change' links for 'Given name' and 'Family name' on the 'Check your answers' page
    • The service navigation doesn’t change the ‘active’ state based on which section the user is in. Currently it’s hardcoded to always highlight ‘Record a grade’, even from the dashboard. The ideal solution to this is probably a combination of layout and include files. I’ve left this out for now due to time constraints.
    • There are links throughout the prototype that have ‘v1’ in their hrefs that ideally would be powered by a sitewide variable instead. It’s something I’ve not implemented personally before, but I know it works. I’ve left it out for now due to time constraints.
